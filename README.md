## Instalación

1. Clona este repositorio: 
```bash
git clone git@github.com:rotgergabriel/losAbuelosReact.git
```
2. Navega al directorio del proyecto: 
```bash
cd losAbuelosReact
```
3. Instala las dependencias: 
```bash
npm install
```
4. Inicia la aplicación: 
```bash
npm start
```
